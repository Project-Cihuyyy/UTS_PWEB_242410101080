## Tentang Web
Aplikasi manajemen toko spare part motor berbasis web menggunakan Laravel. Web ini menampilkan sistem login sederhana dengan fitur dashboard, pengelolaan produk, dan profil pengguna.

## Fitur
- Login dengan username & password
- Dashboard dengan statistik toko
- Pengelolaan daftar produk spare part motor
- Halaman profil pengguna

## Teknologi
- Laravel 13
- Bootstrap 5
- Blade Templating Engine
